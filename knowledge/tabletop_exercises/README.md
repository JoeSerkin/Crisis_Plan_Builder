# Tabletop Exercise Knowledge

Reference materials for designing and facilitating crisis tabletop exercises.

## Exercise design

1. **Objective** — Test a specific capability (activation, comms, escalation), not every plan at once
2. **Scenario** — Realistic injects tied to the client's tier-1 risks and operating context
3. **Participants** — CMT roles with named alternates; include comms and legal where relevant
4. **Facilitation** — Neutral facilitator; pause for decisions before revealing consequences

## Inject cadence

| Time | Inject type |
|------|-------------|
| T+0 | Initial trigger (e.g., site incident, cyber alert, media call) |
| T+15 | Complication (regulator, fatality rumor, supplier failure) |
| T+45 | Resource constraint (executive unavailable, system down) |
| T+90 | Stabilization decision (stand-down vs. escalate) |

## After-action

- Capture gaps against ISO 22361 and client requirements catalog
- Assign owners and dates for corrective actions
- Schedule re-test within 12 months for tier-1 scenarios
