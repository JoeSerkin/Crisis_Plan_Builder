# Incident Management Knowledge

Reference materials for incident classification, situation reporting, and escalation.

## Situation report format (What / So What / What Now)

| Section | Content |
|---------|---------|
| **What** | Facts only — location, time, casualties, operational impact |
| **So What** | Implications for people, operations, reputation, regulators |
| **What Now** | Immediate actions, owners, and next decision point |

## Severity alignment

Incident severity should map to crisis levels and notification matrices:

- **Minor** — Site-managed; CMT not activated
- **Major** — CMT partial or full activation; executive notification
- **Catastrophic** — Full CMT; board/regulator paths per plan

## Consultant prompts

- Is there a single incident logging tool with timestamped entries?
- Who converts an operational incident into a crisis activation?
- Are shift handovers documented so overnight events do not lose context?
