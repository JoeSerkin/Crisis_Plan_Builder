# Crisis Communications Knowledge

Reference materials for internal, external, and stakeholder communications during crises.

## Communication tiers

| Audience | Typical channel | Timing |
|----------|-----------------|--------|
| Employees | SMS, app, manager cascade | Within 1 hour of activation |
| Customers / partners | Email, account teams | After facts verified |
| Media / public | Spokesperson, holding statement | When reputational exposure is material |
| Regulators | Pre-approved templates | Per statutory deadlines |

## Holding statement structure

1. Acknowledge the event without speculation
2. Confirm safety actions taken
3. State that investigation is underway
4. Provide a single point of contact or next update window

## Consultant prompts

- Is there an approved spokesperson hierarchy with trained alternates?
- Are translation resources defined for multi-country employee bases?
- Does social media monitoring feed the crisis comms lead in real time?
